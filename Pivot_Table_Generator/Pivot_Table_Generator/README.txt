***INSTALLATION AND RUNNING INSTRUCTIONS***

To installing the Pivot Table Generator on Windows:
1. Go to /Pivot_Table_Generator folder.
2. Click on GroupC_PivotTable.exe to run on Windows
3. Click on GroupC_PivotTable.jar to run on other platform that has the JRE installed
